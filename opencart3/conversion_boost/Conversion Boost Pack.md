# Conversion Boost Pack — Module Documentation

**Version:** 1.0.0
**Author:** Emanuil
**Compatibility:** OpenCart 3.x

---

## What it does

Conversion Boost Pack enhances product pages by adding two conversion-focused features:

* A **countdown timer** for products with active special offers.
* A **sticky Add to Cart bar** that remains accessible as customers scroll through the product page.

These features are designed to improve visibility of promotions and encourage purchases.

---

## Installation

1. Upload `conversion_boost.ocmod.zip` via **Extensions → Installer**.
2. Go to **Extensions → Modifications** and click **Refresh**.
3. Install **Conversion Boost Pack** from **Extensions → Extensions → Modules**.
4. Open the module settings and enable it.

---

## Configuration

Found at **Extensions → Modules → Conversion Boost Pack**

| Setting         | Description                                        |
| --------------- | -------------------------------------------------- |
| Status          | Enable or disable the module.                      |
| Sticky Bar      | Configure position, colors, and mobile visibility. |
| Countdown Timer | Configure timer style, colors, and display text.   |

---

## How it works

```text
Product has an active Special Price
          ↓
Countdown timer appears automatically
          ↓
Customer scrolls past Add to Cart
          ↓
Sticky Add to Cart bar becomes visible
          ↓
Customer can quickly purchase from anywhere on the page
```

---

## Features

* Automatic countdown timer for active special offers.
* Sticky Add to Cart bar with product information.
* Multiple display and color customization options.
* Mobile visibility controls.
* Uses OpenCart's standard Add to Cart functionality.
* Compatible with OpenCart 3.x.
